# 7️⃣ Orchestrator Agent

## Purpose
The Orchestrator Agent coordinates the entire AI agent system from a single user goal.

It acts as the system controller that connects routing, planning, execution, and memory into one cohesive workflow.

## Responsibilities
- Accept a high-level user request
- Route the task to the correct agent
- Generate a structured execution plan
- Execute the plan step-by-step
- Persist execution results into memory
- Recall relevant past executions
